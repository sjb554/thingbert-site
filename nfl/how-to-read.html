How to Read the NFL Recommendations

Every row on the board is a model-backed wager. Use this cheat sheet to understand the columns, spot the edges, and size your stakes responsibly.

What you are seeing: the table surfaces bets where the model detects positive expected value. EV% shows value while Model Prob shows confidence.

Field Definitions

Field Meaning How to use it
Game Matchup with home team listed second. Group rows to see all plays tied to one game.
Market ML (moneyline), Spread, or Total. ML = winner, Spread = margin, Total = combined points.
Side Direction of the bet. Home/Away for ML or Spread, Over/Under for Totals.
Model Prob Model-implied probability of success. >= 60% usually signals a strong conviction.
Book Odds Quoted sportsbook price in American format. Determines payout and drives implied probability.
Implied Prob Probability implied by the posted odds. American odds: +X -> 100/(X+100), -X -> |X|/(|X|+100).
Edge % Gap between model and market. Edge = ModelProb - ImpliedProb (percentage points).
EV % Expected return per dollar staked. EV% = 100 * (ModelProb * decimalOdds - 1).
Stake $ Recommended stake using fractional Kelly sizing. Treat as a cap and stay within bankroll limits.
Line Spread or total number tied to the bet. Examples: -3.5, +2.5, o46.5, u41.
Book Sportsbook offering the quoted price. Place the bet where the edge was measured.

Quick Interpretation

Signal What it means Recommended action
High EV (>= 5%) + High Prob (>= 60%) Premium value with aligned confidence. Strike quickly if the number is still live.
Moderate EV (2-4%) Edge exists but thinner. Consider when Model Prob is at least 55%.
Low EV (< 2%) Likely breakeven after vig. Monitor for a better price swing.
Negative EV Market has the superior price. Pass or look for alternate angles.

Worked Example

Inputs
  • Market: Moneyline
  • Odds: +115
  • Model Prob: 0.58 (58%)
  • Implied Prob: 100 / (115 + 100) = 0.465 (46.5%)
Outputs
  • Edge %: 0.58 - 0.465 = +0.115 (+11.5 pts)
  • Decimal odds: 1 + 115/100 = 2.15
  • EV %: (0.58 * 2.15 - 1) * 100 = +24.7%
Rule of thumb: focus on plays where both EV% and Model Prob are strong. Use Stake $ as an upper bound, never as a mandate.
